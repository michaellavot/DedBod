# DedBod – התקנה כ־PWA דרך GitHub Pages

## שלבים להעלאה
1. היכנס ל־github.com ופתח חשבון אם אין לך.
2. צור Repository חדש בשם: `DedBod`.
3. העלה לתוכו את כל הקבצים מתוך ה־ZIP הזה:
   - index.html
   - manifest.json
   - sw.js
   - dedbod_icon_192.png
   - dedbod_icon_512.png
   - וכל שאר הקבצים
4. אחרי ההעלאה כנס ל:
   Settings → Pages
5. תחת Build and deployment:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
6. לחץ Save.
7. אחרי דקה-שתיים תקבל קישור בסגנון:
   `https://YOUR_USERNAME.github.io/DedBod/`

## התקנה בטלפון
1. פתח את הקישור בכרום באנדרואיד.
2. לחץ תפריט ⋮.
3. בחר: "התקן אפליקציה" או "הוסף למסך הבית".
4. DedBod תופיע כאפליקציה במסך הבית.

## עדכונים בעתיד
כדי לעדכן את האפליקציה:
1. מחליפים את הקבצים ב־GitHub.
2. מרעננים את האפליקציה בטלפון.
3. אם השינוי לא מופיע מיד, סגור ופתח מחדש את האפליקציה.
