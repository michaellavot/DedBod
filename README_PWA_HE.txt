DedBod PWA

איך להתקין באנדרואיד:
1. העלה את כל הקבצים לתיקייה אחת באחסון/אתר עם HTTPS.
2. פתח את הכתובת בכרום באנדרואיד.
3. לחץ על "התקן כאפליקציה" אם הכפתור מופיע, או תפריט ⋮ → "הוסף למסך הבית".
4. DedBod תופיע במסך הבית עם האייקון שלה ותיפתח כמו אפליקציה.

חשוב:
פתיחה מקובץ מקומי עובדת לבדיקה, אבל התקנה מלאה, Service Worker ועבודה אופליין דורשים HTTPS או localhost.
